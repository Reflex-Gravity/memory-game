Guess Memory

Instructions:

  • You can select the difficulty level.
  • This is a memory game. You need to remember the boxes which are RED.
  • After few seconds all boxes will turn white. Now is the time you need to select the boxes which were red.
  • If you click a box which was BLUE, you lose.
  • There'll be a counter running and you should guess all the red boxes before it finishes.
